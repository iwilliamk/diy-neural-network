# Raw Neural Network (No Open Source Libaries)

This repo contains the implementation of a neural network built from scratch without the use of any open source libraries. I included the complete derivation of the math behind the neural network. It ultimately explores the core concepts of deep learning, including forward propagation, backpropagation, and weight optimization.

## Files Included
* `network.py`: The core neural network architecture and training loop.
* `lab_report.tex`: The LaTeX source code for the mathematical breakdown and final analysis of the model's accuracy.

## How to Run
Ensure you have Python installed.
Run the main script:
   ```bash
   python network.py
